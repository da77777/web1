원시웹

internet vs web의 차이
인터넷이 도시라면, 웹은 도시 위의 건물
인터넷이 도로라면, 웹은 도로 위의 차
인터넷이 운영체제(윈도우)라면, 웹은 운영체제 위의 프로그램

웹은 스위스에서 탄생하였다. 1990년 스위스에 위치한 유럽입자물리연구소에 인터넷이 들어오고, 웹 페이지를 만드는 편집기가 만들어지고, 세계 최초의 웹브라우저인 World Wide Web이라는 프로그램이 만들어졌으며, 직원인 팀 버너스 리가 웹 서버를 만들고 그 서버에 info.cern.ch 라는 주소를 부여한다.

1960년 인터넷이 등장한 이후, 1990년 웹을 만나면서 인터넷이 대중화된다.

인터넷

인터넷이 동작하기 위해선 web browser를 가진 컴퓨터 1대, web server를 가진 컴퓨터 1대로 총 2대의 컴퓨터를 갖고 있어야 한다.
웹브라우저에서 웹서버에 코드를 요청, 웹서버에서 자기가 갖고 있던 코드를 웹브라우저에 보내줌으로써 고객들이 원하는 것이 웹브라우저에 표시된다.
웹브라우저가 깔린 컴퓨터를 client 컴퓨터, 웹서버가 깔린 컴퓨터를 server 컴퓨터라고 한다. 우리는 웹서버를 공부할 것이고, 웹서버를 다룰 줄 안다는 것은 전세계 누구나 내 컴퓨터에 있는 문서를 볼 수 있게 한다는 것이다. 방법은 2가지
web hosting을 통해 업체를 활용하는 것은 쉽지만 배울 수 있는 기회가 적고
web server를 직접 하는 것은 어렵지만 배울기회가 많다.
생활코딩은 웹호스팅을 통해 쉽게 해본 이후에 직접 웹서버를 다뤄보는 것을 추천한다.
